Here I am going to add a new feature that will log all the
queries I make through the database, using "KNEX-LOOGER".
It's important to note that the last version available of
knex-logger was 10 years ago, however, I have used it in other
projects and it worked well... so, I will use it again.

That simple installation command "npm i -S knex-logger" and
wait for it to finish.

Once it's done, just add it to 'app.js' to be used by the
application.

Of course, this won't be turned on forever in the application
because it pollutes the terminal too much, so it's right to use
it when you need to know exactly the query that was made to the
database.

For now, I'll leave it on.
